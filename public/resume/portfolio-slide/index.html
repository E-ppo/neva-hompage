Project 01

AllBarLaw — 법률 플랫폼 (User + Admin)

2025.04 – 2026.01 · 9개월 · 60,000+ TS 라인 단독 개발
납품 완료 / 출시 대기

한 사람이 9개월간 단독 개발한 60,000+ TS 라인 법률 플랫폼. User 코드베이스에서 일반 유저 / 변호사 두 페르소나를 권한 분기로 운영하고, 운영자용 Admin은 별도 코드베이스로 함께 단독 구축. AI 콘텐츠 요약 · 9개 도메인 RBAC · 실시간 채팅까지 단독 책임.

Key Points

  • Socket.io 실시간 채팅(바로톡) — 낙관적 업데이트(tempId), 채팅방별 메시지 캐시, 지수 백오프 재연결.
  • AI 콘텐츠 요약 — 블로그·영상 URL → AI 요약 서버 → 사용자 화면 렌더링. (카운트다운에이아이 fit)
  • React Query / Zustand 분리 아키텍처 — 17개 도메인별 API 서비스 + 63개 커스텀 훅으로 비즈니스 로직 모듈화.
  • 9개 도메인 통합 어드민 + RBAC 동적 메뉴 — 권한 ID 배열 기반, dnd-kit 옵티미스틱 정렬 + 롤백, 두 에디터 분리, xlsx 내보내기.
  • 9개월간 960건 커밋 + GitHub Actions CI/CD 직접 구성 — 단독 운영의 깊이.
